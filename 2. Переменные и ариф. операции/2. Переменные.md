1. Для хранения данных в Go применяются переменные. Тип переменных - var (variable), указывается заранее:
	var - имя_переменной тип_переменной
		var hello string
2. В Go тип переменных может определятся языком автоматически, если не указывать его заранее, но это работает только с такими переменными.
	 0 - для целочисленных типов таких как [[int]]
	 0.0 - для чисел с плавающей точкой [[float]]
	 false - для типа [[bool]]
3. Кроме того, Go позволяет опустить тип пременных при их объявлении и автоматически определить его тип:
	var a = 12
	var hello = "Hello"
4. Примеры объявления переменных:
	 var x int 
	 var y float64
	 var max, min int
	 var hello string
5. Переменные так же можно объявлять по очереди:
	 var a , b , c string
6. Или задать им значение сразу при объявлении:
	 var x int = 10
	 var c string = "Ska bl"
	 var float64 =1.005
