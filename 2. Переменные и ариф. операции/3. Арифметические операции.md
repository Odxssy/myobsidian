1. С переменными можно проводить математические операции:
	+ + сложение
	+ - вычитание
	+ / деление
	+ * умножение
2. При делении чисел будут выводиться только целые значения:
	+ var a int = 10 / 6 , ответ будет 1
3. Для того что бы ответ выдавался с плавающей точкой, нужно правильно указать переменную вещественного типа float64:
	+ var a float64 = 10.0 / 6.0 , указывать значения в таком случае нужно с точкой на конце, в таком случае ответ будет 1.6(6)
4. Что бы вывести остаток от операции, нам нужно использовать вместо арифметических знаков, знак % :
	+ var a int = 10 % 3 , в таком случае ответ будет 1 (10 - 9(3х3))
5. Так же есть инкременты которые добавляют 1 или декремент отнимают: 
	+ var a int = 1
	  a++
	  fmt.Println(a)
	  Вывод : 2
	+ var b int = 1
	  b--
	  fmt.Println(b)
	  Вывод: 0
6. Инкремент и декременты нельзя вписать в переменную, их нужно указывать в разных строках кода:
	+ x++
	  y = x
