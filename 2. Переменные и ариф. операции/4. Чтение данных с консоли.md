Что бы ввести данные в консоль нам нужно обратиться к библиотеке fmt. модулю Scan и указать имя для нашей переменной через &.

Например :

fmt.Scan(&name)

В таком случае консоль спросить у нас пустое поле. Но мы может поставить текст перед командой с помощью знакомой fmt.Println("Как тебя зовут?")

	`package main
	
	import "fmt"
	
	func main () {
		var name string
		var age int
		fmt.Println("Твое имя: ")
		fmt.Scan(&name)
		fmt.Println("Твой возраст: ")
		fmt.Scan(&age)
		
		fmt.Println(name, age)
	}`
Так же можно читать сразу несколько переменных с консоли сразу - fmt.Scan(&a, &b, &c).
