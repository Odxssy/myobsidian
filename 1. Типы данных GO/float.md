Это те числа в которых есть число после запятой.
В отличии от [[complex]], float - содержит только одно значение.
Например 1.123, - 0.000005  и т.д.

**Числа с плавающей точкой невозможно представить точно.** Потому как там имеется мнимое число для исключения бесконечных нерешенностей. 

Размер числа влияет на точность вычислений. То есть вычисления при 32 битах будут менее точны, чем на 64 битах.

**Вещественные числа** - это те же числа с плавающей точкой.
С ними работают **два основных типа**:
	float32 - числа с одинарной точностью.
	float64 - числа с двойной точностью.
Для большей точности используют 64бита, но по умолчанию используется 32бита.
