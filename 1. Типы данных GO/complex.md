Это число с плавающей точкой, почти как [[float]], только может содержать в себе несколько значений.

В complex используется значения real - с реальным числом, и imag (imaginary) - мнимое число.
	z = a + bi  
	%%где i - это мнимое число для решения бесконечного вычисления и избегания ошибки%%

Мнимое число - число добавленное для решения бесконечных вычислений,
например x^2 = -1. Где варианты ответов 1 \ 1 и -1 \ -1 не дадут верного ответа.

В Go есть два основных типа работы с комплексными (числами включающими в себя мнимое) числами:
	complex64
	complex128 - состоящий по умолчанию из двух complex64
